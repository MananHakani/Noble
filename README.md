# Noble

**Noble is a python-based programming language built using the Sly library.**

## Noble Arithmetic Operators

Arithmetic operators are used with numeric values to perform common mathematical :

| Operator | Name           | Example |
| :------: | :------------: | :-----: |
|    +     | Addition       |   x+y   |
|    -     | Subtraction    |   x-y   |
|    *     | Multiplication |   x*y   |
|    /     | Division       |   x/y   |
|    **    | Exponentiation |   x**y  |
|    ***   | Root           |  x***y  |
|    %     | Modulus        |   x%y   |
